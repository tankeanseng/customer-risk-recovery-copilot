# Customer Risk & Recovery Copilot

Customer Risk & Recovery Copilot is an agentic AI decision-support web app for customer payment-risk monitoring, recovery action recommendation, policy-aware approvals, and what-if simulation.

The detailed product, UI, and implementation plan lives in [`docs/`](docs/README.md).

## Planned workspace structure

- `frontend/` - Next.js web app
- `backend/` - FastAPI API and workflow backend
- `shared/` - shared contracts and reference notes
- `data/` - demo data and generated snapshots
- `docs/` - product, design, and implementation specs
